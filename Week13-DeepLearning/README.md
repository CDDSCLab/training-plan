## Project2
- 空气污染预测
- 数据集：[Beijing PM2.5 Data Data Set](https://archive.ics.uci.edu/ml/datasets/Beijing+PM2.5+Data)
- 项目要求：

   2. 自己选择合适的预处理方式
   3. 自己选择合适的深度学习模型
   4. 使用pytorch实现
   5. （选做）分析一天中污染较高的时段
