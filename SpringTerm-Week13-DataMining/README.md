# SpringTerm-Week13-DataMining
## 数据挖掘常用算法：

1. 数据预处理算法
  1. 滤波算法：中值滤波、均值滤波、卡尔曼滤波
  2. 插值算法：最近邻插值、拉格朗日插值
  3. 降维算法：PAC
2. 关联分析算法：Apriori、FP-Growth
3. 图算法
  1. 关键节点：PageRank、k-shell
  2. 社团发现：Girvan Newman、Fast Newman
  3. 频繁子图挖掘：GSpan
4. 序列模式挖掘：PrefixSpan
